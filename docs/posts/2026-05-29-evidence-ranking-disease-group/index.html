1 Research question

Which disease-group (Study_group_new) findings are strongest candidates for future validation?

We synthesize tests already run across the capstone pipeline into a single evidence-ranking matrix, ordered by a transparent score combining p-value and effect size.

2 Data and inputs

Domain Tests included Primary inputs
Mycobiome (beta) PERMANOVA x taxonomic level data/processed/{phylum,family,genus,species}.csv
Mycobiome (alpha) Kruskal-Wallis x diversity metric data/intermediate/alpha_long.rds
Diet Kruskal-Wallis x nutrient data/processed/dietary_cleaned.xlsx + mycobiome metadata
Symptoms Kruskal-Wallis x symptom score data/processed/cleaned_characteristics.csv + mycobiome metadata
  • Comparison: Active IBD vs Quiescent vs Non-IBD (Study_group_new).
  • Unit: Sample-level for mycobiome; participant-level for diet and symptoms (diet/symptoms joined to mycobiome participants).
  • Not included here: Continuous symptom-composition associations (Spearman/PERMANOVA on HBI), stratified nutrient-diversity Spearman, or unsupervised PCA clusters; see linked posts below.

Summary: 24 study-group comparisons ranked; 1 with p < 0.05; 1 tagged High priority.

3 Methods

Item Choice
Integration Recompute or aggregate from processed/intermediate files (same sources as individual posts)
Beta diversity PERMANOVA (adonis2), Bray-Curtis, 999 permutations; effect = R^2
Alpha, diet, symptoms Kruskal-Wallis; effect = eta^2[H] (rstatix::kruskal_effsize)
Evidence score -log10(p) * (0.5 + 0.5 * effect_scaled) with R^2 or eta^2[H] capped at 1
Validation tier High: p < 0.05 and score >= 75th percentile among significant; Medium: p < 0.05; Suggestive: p < 0.10; Low: otherwise
Multiplicity Exploratory ranking; no global FDR across all rows

6 Interpretation

The matrix does not replace individual analysis posts; it prioritizes which disease-group contrasts to carry forward (e.g. pre-registration, larger cohort, or experimental follow-up).

High tier findings combine statistical significance with relatively large R^2 (composition) or eta^2[H] (rank-based outcomes). Low tier rows may still be biologically interesting but need replication. Domains with few participants (especially diet and symptoms) should be interpreted cautiously.
